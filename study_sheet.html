8️⃣ Data Flow (How It Works)

User Opens Website

1. Browser loads http://127.0.0.1:3000/index.html 2. JavaScript executes 3. Fetches GET /api/categories from FastAPI 4. FastAPI returns: ["Electronics", "Clothes", "Shoes", ...] 5. Homepage displays category buttons 6. User clicks category 7. JavaScript fetches GET /api/products/Electronics/Laptops 8. Shows 6 Dell, HP, Lenovo, Asus laptops with prices

User Adds to Cart

1. User clicks "Add to Cart" on Dell Laptop 2. JavaScript sends POST /api/cart/user123 Body: { name: "Dell Laptop", price: 55000, qty: 1 } 3. FastAPI receives request 4. Stores in MongoDB (or in-memory if no DB) 5. Returns updated cart 6. JavaScript displays cart (₹55,000) 7. Cart count updates to "1"

User Checks Out

1. User clicks "Checkout" 2. JavaScript sends POST /api/checkout/user123 3. FastAPI deletes cart from MongoDB 4. Returns success message 5. Shows "Order Placed! ✓" 6. Cart is now empty

1️⃣1️⃣ Deployment Steps

Step 1: GitHub Setup

  1. Create GitHub account
  2. Create new repository "shopease"
  3. Push your code to GitHub

Step 2: Railway Deployment

  1. Go to https://railway.app
  2. Sign up with GitHub
  3. Connect your shopease repository
  4. Railway auto-deploys (watches for changes)
  5. Get your backend URL: https://shopease-api.railway.app

Step 3: GitHub Pages (Frontend)

  1. Create folder "docs" in repository
  2. Move index.html to docs/
  3. Update API URL in index.html: const API = "https://shopease-api.railway.app/api"
  4. Commit and push to GitHub
  5. In repo Settings → Pages → Select "docs" folder
  6. Your site: https://username.github.io/shopease

Step 4: Add Databases

PostgreSQL:
  • Add PostgreSQL to Railway project
  • Copy connection string
  • Set PG_* environment variables
MongoDB:
  • Go to https://www.mongodb.com/cloud/atlas
  • Create free cluster
  • Copy connection string
  • Set MONGO_URI environment variable

1️⃣3️⃣ Key Concepts Explained

❓ What is REST API?

A way for programs to talk to each other using HTTP (web language).

GET /api/products → Ask for data POST /api/cart → Send data to create PUT /api/cart/:id → Send data to update DELETE /api/cart/:id → Request to delete

❓ What is FastAPI?

Modern Python web framework for building APIs quickly and with less code.

❓ What is Uvicorn?

ASGI server that runs Python web applications. Think of it as the "engine" that powers FastAPI.

❓ PostgreSQL vs MongoDB?

PostgreSQL: Strict structure (perfect for products with fixed fields)

MongoDB: Flexible structure (perfect for carts that change)

❓ What is GitHub Pages?

Free website hosting from GitHub for static files (HTML, CSS, JavaScript).

❓ What is Railway?

Cloud platform to host your FastAPI server. Auto-deploys when you push code.
