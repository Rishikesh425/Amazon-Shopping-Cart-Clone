⚡ REST API Reference

# ─── ShopEasy REST API ─────────────────────────────── # Base URL: https://api.shopeasy.example.com/v1 # Auth: Bearer <token> in Authorization header # Content-Type: application/json # ── CATEGORIES ────────────────────────────────────── GET /categories → Returns all top-level categories GET /categories/:category/subcategories → Returns subcategories for a category # ── PRODUCTS ──────────────────────────────────────── GET /products → List all products ?category=Electronics ?subcategory=Laptops ?search=dell ?min_price=5000&max_price=50000 ?in_stock=true ?page=1&limit=20 GET /products/:id → Get single product detail GET /products/:id/stock → Get real-time stock level # ── CART ──────────────────────────────────────────── GET /cart → Get current cart POST /cart/items → Add item Body: { "product_id": "...", "qty": 2 } POST /cart/items/:product_id → Update qty Body: { "qty": 3 } DELETE /cart/items/:product_id → Remove item from cart ?qty=1 (omit to remove all) DELETE /cart → Clear entire cart # ── ORDERS ────────────────────────────────────────── POST /orders → Place order Body: { "name": "Ravi Kumar", "address": "123 MG Road", "payment_method": "cod" | "online", "account_no": "1234567890" // if online } GET /orders/:order_id → Get order status + receipt GET /orders → Order history ?page=1&limit=10 # ── PAYMENT ───────────────────────────────────────── POST /payment/verify → Verify bank account Body: { "account_no": "1234567890" } ← { "name": "Ravi", "balance": 100000 } POST /payment/process → Deduct payment Body: { "account_no": "...", "amount": 55000 } ← { "success": true, "remaining_balance": 45000 } # ── STOCK ─────────────────────────────────────────── POST /stock/reserve → Reserve stock Body: { "product_id":"...", "qty":2 } POST /stock/release → Release reservation # ── EXAMPLE FETCH CALLS ───────────────────────────── // Get all Electronics products: fetch('/v1/products?category=Electronics') .then(r => r.json()).then(data => console.log(data)) // Add Dell Inspiron to cart: fetch('/v1/cart/items', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ product_id: 'dell-inspiron-15', qty: 1 }) }) // Place COD order: fetch('/v1/orders', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ name: 'Ravi Kumar', address: '123 MG Road, Bangalore', payment_method: 'cod' }) }) // Online payment with bank account: fetch('/v1/orders', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ name: 'Anita', address: '45 Park Street, Mumbai', payment_method: 'online', account_no: '9876543210' }) })
